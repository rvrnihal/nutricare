## 🚀 Quick Start

### 1. Load in Server (server.js)

```javascript
const foodsDB = loadFoodsDatabase(); // 88+ foods
const medicinesDB = loadMedicinesDatabase(); // 1,500+ medicines
```

### 2. Lookup Examples

**Food Lookup:**
```javascript
const food = foodsDB.index.get("chicken tikka masala");
// Returns: { name, calories, protein, carbs, fat, region, type, ... }
```

**Medicine Lookup:**
```javascript
const medicine = medicinesDB.index.get("paracetamol");
// Returns: { name, generic, category, uses, dosage, side_effects, ... }
```

### 3. API Endpoints

**GET Food Nutrition:**
```bash
POST /api/analyze-nutrition
Body: { "food": "Biryani" }
```

**GET Medicine Info:**
```bash
POST /api/get-medicine-info
Body: { "medicine": "Aspirin" }
```

### **Method 1: Use the Advanced Home Screen**


### **Method 2: Copy-Paste Implementation Pattern**
Use this in ANY screen:

```dart
import 'package:nuticare/core/design/premium_colors.dart';
import 'package:nuticare/core/design/premium_typography.dart';
import 'package:nuticare/core/animations/premium_animations.dart';
import 'package:nuticare/core/animations/page_transitions.dart';
import 'package:nuticare/widgets/premium_glass_card.dart';
import 'package:nuticare/widgets/animated_gradient_button.dart';

class MyScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // Background color
    return Scaffold(
      backgroundColor: PremiumColors.darkLuxury,  // #0A0E1A
      
      body: SafeArea(
        child: SingleChildScrollView(
          child: Column(
            children: [
              // Animated header
              FadeInAnimation(
                duration: AnimationDurations.standard,  // 400ms
                child: Text(
                  'Your Title',
                  style: PremiumTypography.headlineLarge,
                ),
              ),
              
              // Glass card
              ScaleInAnimation(
                duration: AnimationDurations.standard,
                child: PremiumGlassCard(
                  padding: const EdgeInsets.all(20),
                  child: Text('Content', style: PremiumTypography.bodyMedium),
                ),
              ),
              
              // Gradient button
              AnimatedGradientButton(
                label: 'Click Me',
                gradient: PremiumColors.primaryGradient,
                onPressed: () {
                  // Navigate with animation
                  Navigator.push(
                    context,
                    SlidePageRoute(
                      page: NextScreen(),
                      direction: SlideDirection.up,
                    ),
                  );
                },
              ),
            ],
          ),
        ),
      ),
    );
  }
}
```

## 🔍 TROUBLESHOOTING GUIDE

### **Issue: Colors look different on device**
**Solution**: Device display settings vary. All colors tested for WCAG 2.1 AA+ on standard displays.

### **Issue: Animations feel laggy**
**Solution**: 
- Run in Release mode: `flutter run --release`
- Check device performance
- Ensure 60 FPS in DevTools

### **Issue: Text is too small**
**Solution**: All typography tested for readability. Check device font size settings.

### **Issue: Animation timing feels off**
**Solution**: Use AnimationDurations presets (they're optimized).

### **Issue: Import errors**
**Solution**: Ensure all files are in correct directories:
- `lib/core/design/` for colors and typography
- `lib/core/animations/` for animations
- `lib/widgets/` for components

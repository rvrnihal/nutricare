## 🔧 CUSTOMIZATION EXAMPLES

### Change Primary Color
```dart
// Change in premium_colors.dart
static const Color primaryNeon = Color(0xFF00FF00);  // Change to any color
// All buttons automatically use the new color!
```

### Add Custom Color
```dart
// In premium_colors.dart, add:
static const Color customColor = Color(0xFFABCDEF);
static const LinearGradient customGradient = LinearGradient(
  colors: [Color(0xFFABCDEF), Color(0xFF123456)],
);

// Use it:
PremiumStatsCard(accentColor: PremiumColors.customColor)
```

### Adjust Animation Speed
```dart
// In widget file, change duration:
duration: const Duration(milliseconds: 200),  // Faster
// or
duration: const Duration(milliseconds: 600),  // Slower
```
